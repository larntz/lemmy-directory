Description

A place to discuss and support all things Steam Deck.

Replacement for r/steamdeck_linux.

If it looks like the same content is always at the top, make sure to sort by new. See [this issue](https://github.com/LemmyNet/lemmy/issues/3076) for more details. Same applies to any other sublemmy, sort by new for now to get new content.

As we don't have flairs afaict, you can use these prefixes to indicate what type of post you have made, eg:
[Flair] My post title

The following is a list of suggested flairs:
[Discussion] - General discussion.
[Help] - A request for help or support.
[News] - News about the deck.
[Game] - News / info about a game on the deck.
[Update] - An update to a previous post.
[Meta] - Discussion about this community.

These are not enforced, but they are encouraged.
