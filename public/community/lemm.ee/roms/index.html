Description

A dedicated community of people who are wanting to have a discussion/looking for Roms.


**Allowed**:
Rom discussion, Rom hacks, Rom mods, Emulation, Emulators, Console Modding and anything else related to roms are allowed here.

**NOT ALLOWED**:
Any direct links related to roms are **NOT** allowed. If you want to post a link, encode it in Base64.

How to Encode with Base64:
Visit [https://base64encode.org](https://base64encode.org)

Input the link in the you want to post and copy the code it outputs. You can post that code in this community.

How to decode with Base64: Visit [https://base64decode.org](https://base64decode.org) and input the Base64 code you found here, and it will output the link to the Rom.
