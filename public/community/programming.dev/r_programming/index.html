Description

Please use this as a forum to discuss R, and learn more about it. If you have any questions about how to do specific things in R, this is the place to ask.

## Getting Started

You can [download R here](http://www.r-project.org/).

You can [download RStudio here](http://www.rstudio.com/). RStudio is a powerful and well-supported IDE for R. Other development environment options include Emacs addon [Emacs Speak Statistics](https://ess.r-project.org/) and VSCode.

## Learning resources:
- [R for Data Science](https://r4ds.had.co.nz/) - a good introductory book for learning R. Start here if you're overwhelmed.
- [Big Book of R](https://www.bigbookofr.com/) - collection of more than 500 online books/tutorials covering various aspects of R. Some links are to paid books with previews, but most links are to free online textbooks.
