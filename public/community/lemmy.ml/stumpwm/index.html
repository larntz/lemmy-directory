Description

StumpWM is a tiling window manager for X11 written in Common Lisp. Intended to be the successor to Ratpoison, StumpWM is a hackable, everything-and-the-kitchen-sink window manager. StumpWM strives to be customizable yet visually minimal, with no window decorations, icons, or buttons. StumpWM is hosted on Github.
