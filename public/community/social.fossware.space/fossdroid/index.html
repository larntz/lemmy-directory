#### About
A community for sharing and promoting free / libre and open source software on the Android platform.

This means software you are free to modify and distribute, such as applications licensed under the GNU General Public License, BSD license, MIT license, Apache license, etc. and software that isn't designed to restrict you in any way.

Think of free software as free as in freedom of speech, not free beer.

#### Rules

##### Must relate to Android
Your post must be related to the Android operating system.

##### Mention Source License
When promoting applications, please mention the source license (GPL, MIT, etc.).

##### Link Source Code first
This again is about promoting applications. Please include a link to the source code of the application at the top. If the application is hosted on F-Droid, a single link to the F-Droid page is OK, as it includes all relevant information.

##### No Memes
Please refrain from posting memes in this magazine.

##### Promote your own Software
Developers may promote their own apps and updates. No prior confirmation with the mods is required.

##### Please add Badges to your Posts
Please don't forget to add badges to your posts to help categorize them.
