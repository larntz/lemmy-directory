StumpWM
Thursday, June 15, 2023

StumpWM is a tiling window manager for X11 written in Common Lisp. Intended to be the successor to Ratpoison, StumpWM is a hackable, everything-and-the-kitchen-sink window manager. StumpWM strives to be customizable yet visually minimal, with no window decorations, icons, or buttons. StumpWM is hosted on Github. Relevant links: - [Homepage](https://stumpwm.github.io/) - [Manual (git version)](https://stumpwm.github.io/git/stumpwm-git.html) - [Github Page](https://github.com/stumpwm/stumpwm) - [Subreddit](old.reddit.com/r/stumpwm) - [IRC](irc://irc.libera.chat/#stumpwm) - [Matrix IRC Bridge](https://matrix.to/#/!jYnMNMKsTudWMnQRYW:libera.chat?via=matrix.org&via=libera.chat&via=dhsdevelopments.com) Rules: - Be kind and considerate to others. - All posts must be related to StumpWM in some way.
