The Lean Theorem Prover
Friday, June 16, 2023

Lean is a functional programming language that makes it easy to write correct and maintainable code. You can also use Lean as an interactive theorem prover. Lean programming primarily involves defining types and functions. This allows your focus to remain on the problem domain and manipulating its data, rather than the details of programming. This community is for discussion of the Lean programming language/theorem prover and about mathematics, computer science, type theory and anything else relevant.
